Let's See What Enumerable Methods Can Do

Enumerable#group_by

2/5/14

For this week's exploration of what Ruby has to offer us, I discuss enumerable methods, and more specifically Enumerable#group_by. I was not too familiar what enumerable was referencing until I started learning more and more about Ruby. The Enumerable mixin (as it is described on ruby docs) are a group of methods used to search and sort for "collection classes." The simplest of collection classes are ranges, arrays, and hashes, but a collection class can basically be described as a class where the method #each can be used to iterate through it.

Enumerable#group_by is a method that can take a collection and return a hash based on the results of a code block

Enumerable.group_by { |object| block } => result_hash

Let's take an example where we are trying to sort an array of names by the length of each entry

cohort_arr = ['Alex M.', 'Ben C.', 'Ben M.', 'Carl S.', 'Charles G.', 'Debbie M.', 'Doraly P.', 'Grant Z.', 'Greg C.', 'Harry P.', 'Kenneth G.', 'Lucian K.', 'Nicholas D.', 'Ramon R.', 'Sarena B.', 'Vinh B.] cohort_arr.group_by { |name| name.length } #=> {7=>["Alex M.", "Carl S.", "Greg C.", "Vinh B."], 6=>["Ben C.", "Ben M."], 10=>["Charles G.", "Kenneth G."], 9=>["Debbie M.", "Doraly P.", "Lucian K.", "Sarena B."], 8=>["Grant Z.", "Harry P.", "Ramon R."], 11=>["Nicholas D."]}

We are iterating through our array, cohort_arr. We call each entry object 'name' and are grouping by the name.length. The result is a hash where the keys are the results of name.length, and the values are the names, themselves.
Another example, if we wanted to sort by first letter of each name.

corhort_arr.group_by { |name| name[0] } #=> {"A"=>["Alex M."], "B"=>["Ben C.", "Ben M."], "C"=>["Carl S.", "Charles G."], "D"=>["Debbie M.", "Doraly P."], "G"=>["Grant Z.", "Greg C."], "H"=>["Harry P."], "K"=>["Kenneth G."], "L"=>["Lucian K."], "N"=>["Nicholas D."], "R"=>["Ramon R."], "S"=>["Sarena B."], "V"=>["Vinh B."]}

name[0] is what returns the first letter of the each object, and you can see in our resulting hash that we were able to group by the first letter of each name.

Both of these were fairly simple examples of what you can do with the group_by method, and I am sure that you can get more creative so it can be a more powerful tool. If you have any questions about it, feel free to tweet me @benmorimoto
